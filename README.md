# 🛍️ Product Management System

A Product Management System built with **HTML, CSS, and JavaScript** that allows users to create, manage, update, search, and delete products through a clean and responsive interface. The project focuses on practicing CRUD operations, DOM manipulation, Local Storage, and dynamic data management using vanilla JavaScript.

## 🚀 Live Demo

🔗 https://product-management-system-two-eta.vercel.app/

## ✨ Features

* ➕ Add new products
* ✏️ Update existing products
* 🗑️ Delete a single product
* 🧹 Delete all products
* 🔍 Search by title
* 📂 Search by category
* 💰 Automatically calculate the total price
* 💾 Store data using Local Storage
* 📱 Responsive and user-friendly interface

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Local Storage
